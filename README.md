# Weather App

A simple and responsive Weather Application that provides **real-time weather information** for any location using a third-party weather API.
The app allows users to search weather by city name and also automatically detects the userâ€™s location to display local weather conditions.

## Features:
### Real Time Weather Data
### Location search 
### Automatic geolocation

## API Configuration
To run this project locally:
1. Create an account on **OpenWeatherMap**
2. Generate an API key
3. Replace the  weather_api_key with your key in .env file
